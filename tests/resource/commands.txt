# To install playwright 
npm init playwright@latest

# To execute all test cases
npx playwright test

# To execute all test cases in headed mode
npx playwright test --headed

# To execute all test cases from particular file in headed mode
npx playwright test <filepath> --headed
e.g: npx playwright test D:/playwright1-jan-2026/tests/shriniwas/02-browser-navigation.spec.ts --headed
e.g: npx playwright test /tests/shriniwas/02-browser-navigation.spec.ts --headed

# To view the report
npx playwright show-report

# To execute particular test case from a file
npx playwright test tests/shriniwas/03-login-form.spec.ts --grep "verify login 1"

# To execute test file on particular browser
npx playwright test --project=chromium

# To generate the reports using commands
npx playwright test ./tests/shriniwas/09-reporters.spec.ts --reporter=html
npx playwright test ./tests/shriniwas/09-reporters.spec.ts --reporter=line
npx playwright test ./tests/shriniwas/09-reporters.spec.ts --reporter=list
npx playwright test ./tests/shriniwas/09-reporters.spec.ts --reporter=json
npx playwright test ./tests/shriniwas/09-reporters.spec.ts --reporter=junit
npx playwright test ./tests/shriniwas/09-reporters.spec.ts --reporter=allure-playwright

# To generate the allure report based on allure results
allure generate ./allure-results -o ./allure-report

#To open the generated allure report, use below command
allure open ./allure-report

# You need to install allure-commandline to generate allure-reports
npm install -g allure-commandline

# To check the allure version
allure --version

# To clean the existing report and generate new
allure generate ./allure-results -o ./allure-report --clean

# To open a codegen window
npx playwright codegen
# To open a codegen window with file creation
npx playwright codegen -o <filepath>
npx playwright codegen --output <filepath>
# To open a codegen window with lang
npx playwright codegen --target java
# To open a codegen window with browser
npx playwright codegen -b <chromium/firefox/webkit>
npx playwright codegen --browser <chromium/firefox/webkit>
#IMP command: To open our script in debugger window
npx playwright test <filepath> --debug

# Run tests using tags
npx playwright test /tests/shriniwas/25-tags2.spec.ts --grep "@regression"

Works as an AND operator, it will execute the tests which has both the tags
npx playwright test /tests/shriniwas/25-tags2.spec.ts --grep "(?=.*@sanity)(?=.*@regression)"

Works like a Logical OR  
npx playwright test tests/shriniwas/25-tags2.spec.ts --grep "@sanity|@regression"
